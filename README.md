# Full Stack App with React and a REST API

A full-stack JavaScript app built with React and a RESTful API, featuring authentication, protected routes, and robust error handling.

## Features

- Browse and view courses
- User authentication (Sign Up / Sign In / Sign Out)
- Create, update, and delete courses (owner-only access)
- Protected routes and authorization checks
- User-friendly error pages 

## Tech Stack

- Front End: React, React Router, JavaScript (ES6+)
- Back End: Node.js, Express, Sequelize, SQLite
- Auth: Basic Authentication

## Getting Started
`API
cd api
npm install
npm start
`
## Client
`cd client
npm install
npm run dev`

Client runs on http://localhost:5173
API runs on http://localhost:5000
